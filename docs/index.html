Introduction

gitbook 构建方法

安装 Node.js 和 npm

https://nodejs.org/en/download/

安装 GitBook 命令行工具

npm install gitbook-cli -g

下载项目

git clone xxx_project

构建 GitBook

初始化项目(可选)

gitbook init xxx_project

进入项目

cd xxx_project

初始化GitBook属性定义(可选)

vim book.json

进入项目，执行构建

rm -rf docs && gitbook build . docs
